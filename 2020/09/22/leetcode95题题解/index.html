给定一个整数 n，生成所有由 1 … n 为节点所组成的 二叉搜索树 。

示例：

输入：3
输出：
[[1,null,3,2],[3,2,null,1],[3,1,null,null,2],[2,1,3],[1,null,2,null,3]]
解释：
以上的输出对应以下 5 种不同结构的二叉搜索树：

1 3 3 2 1
\ / / / \
3 2 1 1 3 2
/ / \
2 1 2 3

提示：

0 <= n <= 8

这个题解从如何构建二叉树开始，一步步教怎么枚举所有的可能性。可以学会构建二叉树和穷举可能性这两种方法。
